# Revue Finance — Bibliometric Citation Network

Various bibliometric network visualizations for **Revue Finance** (RF) [ISSN 0752-6180], based on Google Scholar data extracted via Harzing's Publish or Perish (PoP).

The visualizations cover the **21 most cited papers published in Revue Finance (2008–2023)** and their citation networks.
